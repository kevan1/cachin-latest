# Product Overview

Cachin is a multi-chain mobile wallet application built with React Native and Expo. The app provides secure wallet management and transaction capabilities across multiple blockchain networks.

## Core Features

- **Multi-chain Support**: Native support for Solana and Monad (EVM-compatible) blockchains
- **Embedded Wallets**: Privy-powered embedded wallets with passkey authentication
- **Cross-chain Balance Tracking**: Unified view of assets across all supported chains
- **Mobile-first UX**: Native iOS and Android experience with offline support
- **Secure Authentication**: Apple Sign-In, passkeys, and biometric authentication

## Key User Flows

1. **Onboarding**: Passkey-based authentication with automatic wallet creation
2. **Balance Management**: Real-time balance tracking with USD conversion
3. **Transactions**: Send/receive tokens across supported chains
4. **Multi-chain Operations**: Seamless switching between Solana and EVM chains

## Target Platforms

- iOS (minimum deployment target: 17.5)
- Android (compile SDK version: 35)
- Development via Expo Go and custom dev client
